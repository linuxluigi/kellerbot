Glossar
=======

.. glossary::
    :sorted:



    GPIO
        Abkürzung für `General Purpose Input Output`, dies bezeichnet eine programmierbare Ein- und Ausgänge für allgemeine Zwecke

    Raspberry Pi
        Ein Open Hardware Minicomputer der entwickelt wurde mit dem Ziel nicht über 35$ zu kosten und der sich durch die
        geringen kosten, geringen Stromverbrauch & durch Open Source Betriebsysteme für viele IT Projekte bewährt und
        beliebt gemacht hat.

    Arduino
        Ein Open Source Hard- & Software Gerät mit einem einfachen Eingabe und Ausgabe Board mit einem
        Mikrocontroller und analogen und digitalen Ein- und Ausgängen. Arduino soll eine einfache PLattform für
        Entwickler Einsteiger sein, die mit dieser Plattform programmieren und den umgang mit Hardware lernen können.

    Arduino Nanos
    Arduino Nano
        Ist eine kompakte Version eines Arduinos.

    DHT22 Temperatur- und Luftfeuchtigkeitssensor
        Der DHT22 ist ein Sensor zur auslesen von Temperatur & Luftfeuchtigkeit, welcher leicht von einen Arduino,
        Raspberry Pi oder ähnlichen Geräten verwendet werden kann.

    Einplatinenrechner
        Auf einer Leiterplatte alle nötigen elektronischen Komponenten zum betreiben eines Computersystemes.

    WLAN Access Point
    Access Point
        Basisstation zur kabellosen kommunikation im LAN.

    openWrt
        Open Source Router Betriebsystem welches meist mehr Funktionen bietet als die Betriebsysteme der Hardware Hersteller
        vorinstallierten Betriebsysteme. OpenWart wird aktiv von Freifunkern verwendet um ein freies offenen Internet
        für alle anbieten zu können.

    WLAN zu LAN Bridge
        Ein WLAN Router welcher sich in ein exitierendes WLAN Netzwerk einwählt und dies als Internet Quelle dem LAN freigibt.

    Token
        Ein Schlüssel zur Authefizierung einer Schnittstelle.

    Powerline
        Ethernet über die Steckdose.
